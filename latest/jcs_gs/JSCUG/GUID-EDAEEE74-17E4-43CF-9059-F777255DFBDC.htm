6 Backing Up and Restoring an Oracle Java Cloud Service Instance on Oracle Public Cloud Machine

You can back up and restore your Oracle Java Cloud Service instances to return their software and data to a particular state.

Note:

This section does not apply to Oracle Java Cloud Service—Virtual Image instances. Backup and restoration are not supported by Oracle Java Cloud Service—Virtual Image instances.
