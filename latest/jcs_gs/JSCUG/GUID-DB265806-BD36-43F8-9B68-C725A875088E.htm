To manage the life cycle of Oracle Java Cloud Service instances, consider the typical workflow described in the following table.

The table provides links to information about how to perform each task by using the web-browser-based Oracle Java Cloud Service Console. For information about using the REST API to manage the life cycle of Oracle Java Cloud Service instances, see Service Instances in REST API for Oracle Java Cloud Service.
