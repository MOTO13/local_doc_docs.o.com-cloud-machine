Deleting the Coherence Data Tier of an Oracle Java Cloud Service Instance on OPCM

You cannot use the Oracle Java Cloud Service Console to remove the Coherence data tier. You can perform the task by using the REST API only.

Only specific resources and entities related to Oracle Coherence in Oracle Java Cloud Service are deleted when you remove the Coherence data tier cluster of storage-enabled Managed Servers from a service instance. A scale in operation is added to the Activity page.

To use the REST API to remove the Coherence data tier, see Delete a Coherence Data Tier Cluster in REST API for Oracle Java Cloud Service.
