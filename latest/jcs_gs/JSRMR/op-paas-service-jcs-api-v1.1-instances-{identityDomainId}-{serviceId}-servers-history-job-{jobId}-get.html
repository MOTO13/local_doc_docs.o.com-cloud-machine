View the Status of a Server Operation by Job ID

get

/paas/service/jcs/api/v1.1/instances/{identityDomainId}/{serviceId}/servers/history/job/{jobId}

Returns information about the server operation performed on the Oracle Java Cloud Service instance based on its job ID.
  • Request
  • Response
  • Examples

Request

Path Parameters
identityDomainId
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

jobId
Type: string
Required: true
Job ID. To view the job IDs for all operations, see View All Server Operations.
serviceId
Type: string
Required: true
Name of the Oracle Java Cloud Service instance.

Type: string
Required: true

Examples

The following example shows how to use the job ID to view information about a server operation for an Oracle Java Cloud Service instance. The example uses cURL to submit a GET request on the REST resource.

Note: The command in this example uses the URL structure https://rest_server_url/resource-path, where rest_server_url is the Oracle Public Cloud or Oracle Cloud Machine REST server to contact for your identity domain or tenant name. See Send Requests for the appropriate URL structure to use on Oracle Public Cloud or Oracle Cloud Machine.

cURL Command

curl -X GET -u username:userPassword1! -H "X-ID-TENANT-NAME:ExampleIdentityDomain" https://rest_server_url/paas/service/jcs/api/v1.1/instances/ExampleIdentityDomain/ExampleInstance/servers/history/job/8971
