Obtain Account Information

From your account administrator, obtain the appropriate credentials that allow you to access Oracle Java Cloud Service to create and manage service instances.

On Oracle Cloud Machine, get the sign-in credentials from your Oracle Cloud Administrator or Tenant Administrator.

You will need:

  • User name and password

  • Identity domain ID (on Oracle Public Cloud)

    If your account administrator has changed the identity domain name, you must still specify the identity domain ID when using the REST API. For example, if the original identity domain name was MyIdentityDomain54321 and it was changed to MyIdentityDomain, you must use MyIdentityDomain54321 in the REST endpoints.

  • Tenant name (on Oracle Cloud Machine)
