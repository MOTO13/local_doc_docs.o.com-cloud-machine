Access Rules REST Endpoints

Access Rules
View and manage network security rules for your Oracle Java Cloud Service instance virtual machines.

These endpoints are only available on Oracle Public Cloud.

Add an Access Rule
Method: post
Path: /paas/api/v1.1/instancemgmt/{identityDomainId}/services/jaas/instances/{serviceId}/accessrules
Update an Access Rule
Method: put
Path: /paas/api/v1.1/instancemgmt/{identityDomainId}/services/jaas/instances/{serviceId}/accessrules/{ruleName}
View All Access Rules
Method: get
Path: /paas/api/v1.1/instancemgmt/{identityDomainId}/services/jaas/instances/{serviceId}/accessrules
