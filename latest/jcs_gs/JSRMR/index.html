About the REST APIs

Describes how to use the Oracle Java Cloud Service REST API to create and manage WebLogic Server instances on both Oracle Public Cloud and Oracle Cloud Machine, unless otherwise indicated.

APIs

Service Instances

Manage the life cycle of your Oracle Java Cloud Service instances.
View API Endpoints

Managed Servers

View information about Managed Servers and their operations for your Oracle Java Cloud Service instances.
View API Endpoints

Scaling

Scale clusters and nodes on-demand in your Oracle Java Cloud Service instances.
View API Endpoints

Automatic Scaling

Manage automatic (auto) scaling out and scaling in of clusters in your Oracle Java Cloud Service instances.

These endpoints are only available on Oracle Public Cloud.

View API Endpoints

Coherence Data Tiers

Manage the Coherence data tiers and capacity units for your Oracle Java Cloud Service instances.
View API Endpoints

Backups and Restorations

Manage backup and restore operations on your Oracle Java Cloud Service instances, including updating the backup configuration, and disabling or enabling the backup service for a service instance.
View API Endpoints

Patches

Patch your Oracle Java Cloud Service instances.
View API Endpoints

SSH Access

View and manage SSH keys for your Oracle Java Cloud Service instances.
View API Endpoints

Access Rules

View and manage network security rules for your Oracle Java Cloud Service instance virtual machines.

These endpoints are only available on Oracle Public Cloud.

View API Endpoints

AppToCloud Migration

Import on-premises WebLogic Server domain resources into your Oracle Java Cloud Service instances.

These endpoints are only available on Oracle Public Cloud.

View API Endpoints

Health Monitoring

Monitor the health of your Oracle Java Cloud Service instances.
View API Endpoints
