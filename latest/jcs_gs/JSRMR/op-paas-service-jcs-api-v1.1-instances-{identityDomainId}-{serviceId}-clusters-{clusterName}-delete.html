Delete a Coherence Data Tier Cluster

delete

/paas/service/jcs/api/v1.1/instances/{identityDomainId}/{serviceId}/clusters/{clusterName}

Deletes the entire Coherence data tier cluster (the storage-enabled WebLogic Server cluster) of an Oracle Java Cloud Service instance.

Note: The term scaling unit used in the REST API means the same as capacity unit.

  • Request
  • Response
  • Examples

Request

Path Parameters
clusterName
Type: string
Required: true
Name of the Coherence data tier cluster to delete from the service instance.
identityDomainId
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

serviceId
Type: string
Required: true
Name of the Oracle Java Cloud Service-Coherence instance.
Header Parameters
X-ID-TENANT-NAME
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

Response

Supported Media Types
  • application/json
202 Response
Accepted. See Status Codes for information about other possible HTTP status codes.
400 Response
Bad Request. Returned if one of the following statements is true:
  • The data tier WebLogic Server cluster does not exist in the domain. The system manager returns the following message: Cluster [DataTierClusterName] not found.
  • A second request is received.

Examples

The following example shows how to delete the Coherence data tier cluster from an Oracle Java Cloud Service instance by submitting a DELETE request on the REST resource using cURL.

Note: The command in this example uses the URL structure https://rest_server_url/resource-path, where rest_server_url is the Oracle Public Cloud or Oracle Cloud Machine REST server to contact for your identity domain or tenant name. See Send Requests for the appropriate URL structure to use on Oracle Public Cloud or Oracle Cloud Machine.

cURL Command

curl -i -X DELETE -u username:userPassword1! -H "X-ID-TENANT-NAME:ExampleIdentityDomain" https://rest_server_url/paas/service/jcs/api/v1.1/instances/ExampleIdentityDomain/ExampleInstance/clusters/ExampleDGCluster
