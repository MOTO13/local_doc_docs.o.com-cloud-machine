Managed Servers REST Endpoints

Managed Servers
View information about Managed Servers and their operations for your Oracle Java Cloud Service instances.
View a Managed Server
Method: get
Path: /paas/service/jcs/api/v1.1/instances/{identityDomainId}/{serviceId}/servers/{name}
View All Managed Servers
Method: get
Path: /paas/service/jcs/api/v1.1/instances/{identityDomainId}/{serviceId}/servers
View All Server Operations
Method: get
Path: /paas/service/jcs/api/v1.1/instances/{identityDomainId}/{serviceId}/servers/history
View the Status of a Server Operation by Job ID
Method: get
Path: /paas/service/jcs/api/v1.1/instances/{identityDomainId}/{serviceId}/servers/history/job/{jobId}
View the Status of a Server Operation by Operation ID
Method: get
Path: /paas/service/jcs/api/v1.1/instances/{identityDomainId}/{serviceId}/servers/history/{id}
