View All Managed Servers

get

/paas/service/jcs/api/v1.1/instances/{identityDomainId}/{serviceId}/servers

Returns information about all Managed Servers for the Oracle Java Cloud Service instance.
  • Request
  • Response
  • Examples

Request

Path Parameters
identityDomainId
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

serviceId
Type: string
Required: true
Name of the Oracle Java Cloud Service instance.
Header Parameters
X-ID-TENANT-NAME
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

Response

Supported Media Types
  • application/json

OK. See Status Codes for information about other possible HTTP status codes.

Properties
clusterName
Type: string
Name of the cluster to which the Managed Server belongs.
creationDate
Type: string
Date and time the server was created.
hostname
Type: string
Host name of the server.
isAdmin
Type: boolean
Flag that specifies whether this Managed Server is also the Administration Server.
jobId
Type: string
Job ID for a scale operation.
name
Type: string
Name of the server.
nodeType
Type: string
Type of node. For example, WLS or DG.
reservedIp
Type: string
This attribute is only available on Oracle Cloud Machine. It is no longer in use.
reservedIpAddress
Type: string
This attribute is only available on Oracle Public Cloud. It is displayed only when the service instance uses an Oracle Exadata database.

IP address for the IP reservation that is assigned to the Managed Server.

reservedIpName
Type: string
This attribute is only available on Oracle Public Cloud. It is displayed only when the service instance uses an Oracle Exadata database.

Name of the IP reservation that is assigned to the Managed Server.

shape
Type: string
Compute shape of the server.

On Oracle Public Cloud, see About Machine Images and Shapes in Using Oracle Compute Cloud Service (for PaaS).

status
Type: string
Status of the operation. For example, Ready.
storageAllocated
Type: string
Amount of allocated storage in MBs.

Examples

The following example shows how to view information about all Managed Servers for an Oracle Java Cloud Service instance by submitting a GET request on the REST resource using cURL.

Note: The command in this example uses the URL structure https://rest_server_url/resource-path, where rest_server_url is the Oracle Public Cloud or Oracle Cloud Machine REST server to contact for your identity domain or tenant name. See Send Requests for the appropriate URL structure to use on Oracle Public Cloud or Oracle Cloud Machine.

cURL Command

curl -i -X GET -u username:userPassword1! -H "X-ID-TENANT-NAME:ExampleIdentityDomain" https://rest_server_url/paas/service/jcs/api/v1.1/instances/ExampleIdentityDomain/ExampleInstance/servers
