View a Deleted Backup

get

/paas/api/v1.1/instancemgmt/{identityDomainId}/services/jaas/instances/{serviceId}/deletedbackups/{jobId}

Returns information about a deleted backup for an Oracle Java Cloud Service instance based on the job ID provided.

On Oracle Public Cloud, see About Backup and Restoration of Oracle Java Cloud Service Instances in Using Oracle Java Cloud Service for complete information about contents of backups, where backups are stored, and how backups and local copies of backups are deleted automatically. On Oracle Cloud Machine, see About Backup and Restoration of Oracle Java Cloud Service Instances in Using Oracle Java Cloud Service on OPCM.

Note: The legacy path /paas/service/jcs/api/v1.1/instances/{identityDomainId}/{serviceId}/deletedbackups is deprecated but it will continue to be supported for the near term. However, it is recommended that you migrate your applications to use the new path at your earliest convenience to minimize any disruption in the future when the deprecated URL structure is no longer supported.

Request

Path Parameters
identityDomainId
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

jobId
Type: string
Required: true
Job ID for the backup deletion. To retrieve the backup deletion job ID, see View All Deleted Backups. This can also be the job ID for the backup deletion (asynchronous) as described in Delete a Backup (Asynchronous).
serviceId
Type: string
Required: true
Name of the Oracle Java Cloud Service instance.
Header Parameters
X-ID-TENANT-NAME
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

Response

200 Response
OK. See Status Codes for information about other possible HTTP status codes.
Body
The response body contains information about a backup deleted synchronously or asynchronously, or a list of backups deleted asynchronously.

Items
  • Type: object delete-backup
Nested Schema : delete-backup
Properties
completeDate
Type: string
Date and time the delete backup operation completed.
jobId
Type: string
Job ID for the delete backup operation.
operation
Type: string
Type of operation that was executed. Set to delete.
startDate
Type: string
Date and time the delete backup operation started.
status
Type: string
Status of the delete backup operation. Valid values include: Completed, Failed, Deleted, or Delete In Progress.
statusDetails
Type: string
History of system messages. Each message is separated by an ellipses {...}.
400 Response
Bad Request. Returned if the job ID does not exist.

Examples

The following example shows how to view information about a deleted backup for an Oracle Java Cloud Service instance by submitting a GET request on the REST resource using cURL.

Note: The command in this example uses the URL structure https://rest_server_url/resource-path, where rest_server_url is the Oracle Public Cloud or Oracle Cloud Machine REST server to contact for your identity domain or tenant name. See Send Requests for the appropriate URL structure to use for Oracle Public Cloud or Oracle Cloud Machine.

cURL Command

curl -i -X GET -u username:userPassword1! -H "Content-Type:application/json" -H "X-ID-TENANT-NAME:ExampleIdentityDomain" https://rest_server_url/paas/api/v1.1/instancemgmt/ExampleIdentityDomain/services/jaas/instances/ExampleInstance/deletedbackups/4779
