View the SSH Key Value

get

/paas/api/v1.1/instancemgmt/{identityDomainId}/services/jaas/instances/{serviceId}/credentials/{credentialName}

Returns the credential value for the given credential name.

Note: vmspublickey is the only supported credential name.

  • Request
  • Response
  • Examples

Request

Path Parameters
credentialName
Type: string
Required: true
Credential name associated with the SSH public key. Currently, the only credential name you can use is vmspublickey.
identityDomainId
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

serviceId
Type: string
Required: true
Name of the Oracle Java Cloud Service instance.
Header Parameters
X-ID-TENANT-NAME
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

Response

Supported Media Types
  • application/json
200 Response
OK. See Status Codes for information about other possible HTTP status codes.
Body
The response body contains details about the SSH public key value.
Root Schema : sshkeyvalue-response
The response body contains details about the SSH public key value.
Properties
componentType
Type: string
Type of service component. Possible values include: WLS, OTD, DB.
computeKeyName
Type: string
Compute sshkey object key name. This attribute is only available on Oracle Public Cloud.
credName
Type: string
Credential name. Currently, the only supported credential name is vmspublickey.
credType
Type: string
Credential type. Only supported value is SSH.
description
Type: string
Credential description of the given credential name.
identityDomain
Type: string
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

lastUpdateMessage
Type: string
Message returned from the last SSH key update. For example: newly created, updated by username.
lastUpdateStatus
Type: string
Status of the last SSH key update.
lastUpdateTime
Type: string
Date and time the SSH key was last updated.
osUserName
Type: string
Virtual machine OS user name. Only supported value is opc.
parentType
Type: string
Valid value is SERVICE.
publicKey
Type: string
Public key for the secure shell (SSH). This key is used for authentication when connecting to the Oracle Java Cloud Service instance using an SSH client.
serviceName
Type: string
Name of the Oracle Java Cloud Service instance.
serviceType
Type: string
Cloud service type. Valid value is jaas for Oracle Java Cloud Service.

Examples

The following example shows how to view the SSH public key value for the given credential name by submitting a GET request on the REST resource using cURL.

Note:

  • Currently, the only supported credential name is vmspublickey.

  • The command in this example uses the URL structure https://rest_server_url/resource-path, where rest_server_url is the Oracle Public Cloud or Oracle Cloud Machine REST server to contact for your identity domain or tenant name. See Send Requests for the appropriate URL structure to use for Oracle Public Cloud or Oracle Cloud Machine.

cURL Command

curl -i -X GET -u username:userPassword1! -H "X-ID-TENANT-NAME:ExampleIdentityDomain" https://rest_server_url/paas/api/v1.1/instancemgmt/ExampleIdentityDomain/services/jaas/instances/ExampleInstance/credentials/vmspublickey
