View the Status of an Add SSH Key Operation by Job Id

get

/paas/api/v1.1/activitylog/{identityDomainId}/job/{jobId}

Returns information about the Add SSH Key operation performed by viewing the status of the Oracle Java Cloud Service operation based on the job ID.
  • Request
  • Response
  • Examples

Request

Path Parameters
identityDomainId
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

jobId
Type: string
Required: true
Job ID of the operation.

Type: string
Required: true

Examples

The following example shows how to use the job ID to view information about the add SSH key operation being performed on the Oracle Java Cloud Service instance. The example uses cURL to submit a GET request on the REST resource.

Note: The command in this example uses the URL structure https://rest_server_url/resource-path, where rest_server_url is the Oracle Public Cloud or Oracle Cloud Machine REST server to contact for your identity domain or tenant name. See Send Requests for the appropriate URL structure to use for Oracle Public Cloud or Oracle Cloud Machine.

cURL Command

curl -i -X GET -u username:userPassword1! -H "X-ID-TENANT-NAME:ExampleIdentityDomain" "https://rest_server_url/paas/api/v1.1/activitylog/ExampleIdentityDomain/job/8722"

Example of Response Header

The following shows an example of the response header.

Date: Tue, 19 Apr 2016 22:01:27 GMT
Transfer-Encoding: chunked
Content-Type: application/json
