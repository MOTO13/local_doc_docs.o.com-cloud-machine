View the Status of a Service Instance Lifecycle Operation

get

/paas/service/jcs/api/v1.1/instances/{identityDomainId}/status/{requestName}/job/{jobId}

Returns the status of an Oracle Java Cloud Service instance lifecycle operation.
  • Request
  • Response
  • Examples

Request

Path Parameters
identityDomainId
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

jobId
Type: string
Required: true
Job ID.
requestName
Type: string
Required: true
Name of the request for which you want to view status. Valid values include:
  • create: View the status for a create service instance operation (Create a Service Instance).
  • delete: View the status of a delete instance operation (Delete a Service Instance).
  • control: View the status of a lifecycle state control operation (Stop and Start a Service Instance and Individual VMs).
  • scale: View the status of a Coherence data tier scale operation (Scale a Coherence Data Tier and Add a Coherence Data Tier Cluster).
Header Parameters
X-ID-TENANT-NAME
Type: string
Required: true
Identity domain ID for the Oracle Java Cloud Service account (on Oracle Public Cloud).

Tenant name for the Oracle Java Cloud Service instance (on Oracle Cloud Machine).

Properties
auto_update
Type: boolean
Flag that specifies whether updates to the Oracle Cloud Tools are automatically applied to the Oracle Java Cloud Service instance during the maintenance window. The Oracle Cloud Tools are used to manage the lifecycle of your service instance.
compliance-status
Type: string
Status indicating whether the version of Oracle Cloud Tools is out of compliance. Oracle uses Oracle Cloud Tools to manage the lifecycle of your service instance.

If the Oracle Cloud Tools are out of compliance, this attribute is set to one of the following status values:

  • NEW_VERSION: Indicates a new version of Oracle Cloud Tools is available. Oracle strongly recommends that you apply this update as soon as possible.
  • DEPRECATED: Indicates that the current version of Oracle Cloud Tools is deprecated. Apply the latest Oracle Cloud Tools update to avoid any disruption of service in the future.
  • UNSUPPORTED: Indicates that the current version of Oracle Cloud Tools is not supported. Apply the latest patch to resume normal operations.

If the Oracle Cloud Tools are up-to-date, this attribute is blank.

compliance_status_desc
Type: string
Description that provides more details about the compliance status of the Oracle Cloud Tools, used to manage the lifecycle of your Oracle Java Cloud Service instance. If the service instance is out of compliance, this attribute is set to one of the following descriptions, based on the status value:If the Oracle Cloud Tools are out of compliance, this attribute is set to one of the following status values:
  • NEW_VERSION: newer version of Oracle tools latestVersion is available. This update includes critical fixes to Oracle Cloud Tools. Oracle uses cloud tools to manage lifecycle of your service. Oracle strongly recommends that customers apply this update as soon as possible.
  • DEPRECATED: This service is currently in a deprecated state because the Oracle tools version deprecatedVersion is deprecated. Apply the latest Oracle tools update as this version may not be supported in the future.
  • UNSUPPORTED: This service is currently in an unsupported state because the Oracle tools version unsupportedVersion is obsolete. Apply the latest patch to resume normal operations.

If the Oracle Cloud Tools are up-to-date, this attribute is blank.

Examples

The following example shows how to view the status of an Oracle Java Cloud Service instance operation by submitting a GET request on the REST resource using cURL. The REST resource URI is returned in the Location header when submitting the service instance request.

Note: The command in this example uses the URL structure https://rest_server_url/resource-path, where rest_server_url is the Oracle Public Cloud or Oracle Cloud Machine REST server to contact for your identity domain or tenant name. See Send Requests for the appropriate URL structure to use on Oracle Public Cloud or Oracle Cloud Machine.

cURL Command

The following command returns the status of a service instance create operation. Other operations are delete, scale, and control.

curl -i -X GET -u username:userPassword1! -H "X-ID-TENANT-NAME:ExampleIdentityDomain" https://rest_server_url/paas/service/jcs/api/v1.1/instances/ExampleIdentityDomain/status/create/job/7495

Example of Response Header

The following shows an example of the response header for a create operation that has completed :
