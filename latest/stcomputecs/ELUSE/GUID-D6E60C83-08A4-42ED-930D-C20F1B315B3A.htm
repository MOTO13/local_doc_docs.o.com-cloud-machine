Exploring the User Interface

To log into the web console for accessing and monitoring Oracle Compute Cloud Service on OPCM, see Accessing the Self Service Portal in the UI.

The Self Service Portal provides a dashboard to request and manage resources as shown in the following figure.


Description of the illustration GUID-746947B2-AD0C-43CB-B95F-F187C841130C-default.png

Home Page

The Home page displays the existing orchestrations and statuses. The page provides the options to discover, create, and manage orchestrations.


Description of the illustration GUID-1BEB9F07-F47D-4E3E-8984-59B294A4ADB9-default.png

Networks Page

The Networks page displays the tenant networks and service networks that are available for use, and IP addresses that were reserved in these networks.


Description of the illustration GUID-740C8C80-1DDD-4429-93FE-71E9F56FB4AD-default.png

Storage Page

The Storage page displays all the storage volumes in the tenant. The page provides options to create and manage storage volumes and snapshots.


Description of the illustration GUID-C8C17F60-7715-4E28-AA4C-F9CF4FC3401E-default.png

Library Page

The Library page displays all the templates and deployment plans saved for later use. The page provides options to upload and manage private templates, and view saved deployment plans.
