The image is a diagram that outlines the process of creating and accessing an instance in Oracle Compute Cloud Service.

The three user roles depicted in the illustration are:

  • Oracle Cloud Administrator

  • Tenant Administrator

  • Tenant User

The following steps are listed in the illustration:

  • Step 1: Create Tenants

  • Step 2: Create Service Networks - EoIB and IPoIB

  • Step 3: Create Tenant Networks - EoIB and IPoIB

  • Step 4: Share Service Networks with Tenants

  • Step 5: Create Shapes

  • Step 6: Manage Public Templates

  • Step 7: Create Users and Assign Tenant Administrator or Tenant User Roles

  • Step 8: Manage Private Templates

  • Step 9: Create Tenant Networks IPoIB

  • Step 10: Create Storage Volumes

  • Step 11: Create and Manage SSH Keys

  • Step 12: Create and Manage Orchestrations

  • Step 13: Manage Instances

  • Step 14: Access Instances

Step 1 to 6 are done by Oracle Cloud Administrator.

Step 6 can be done by both the Oracle Cloud Administrator and Tenant Administrator.

Step 7 to 14 can be done by both the Tenant Administrator and Tenant User.
