The image is a diagram that outlines the life cycle of an instance. When you start an orchestration, the VM instances are prepared, initialized and launched. An instance is in running state when it is launched. To update an instance, you must shut down the instance. You can delete an instance which is in running or shutdown state. You can restart an instance in shutdown state and move it to running state.
