The image is a screen capture that shows the Preference window. The Infrastructure - Oracle Compute option is selected for the user to set the credentials for the Oracle Compute Site. There are two regions in Infrastructure - Oracle Compute: Oracle Compute Site Credential and Configure SSH Keys and Host Credentials. In the Oracle Compute Site Credential region, you can set credentials for each Oracle Compute Site Name.
