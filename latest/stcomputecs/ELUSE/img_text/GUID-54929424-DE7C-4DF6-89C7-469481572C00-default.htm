The image is a diagram that outlines the lifecycle of an orchestration. When you add an orchestration, the status of the orchestration is stopped. Start the orchestration to move it to the ready status. You can shut down an orchestration which is in ready status. You can stop an orchestration which is in shut down or ready status. When you stop an orchestration, all the VM instances defined in the orchestration are deleted.

You can update or delete an orchestration which is in stopped status.
