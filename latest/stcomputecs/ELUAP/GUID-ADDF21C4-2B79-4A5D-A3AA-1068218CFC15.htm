5 User Management

A user is an entity within the system that is able to make requests. Each user is associated with one tenant, and has an associated role. The user's role determines the actions that the user is allowed to take. Tenant administrators manage users within their tenancy. Users have the /tenant_name/users role by default. Only the Oracle Cloud Administrator can grant a user the /tenant_name/admin role.
