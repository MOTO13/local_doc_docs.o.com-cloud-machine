9 Storage Management

Oracle Compute Cloud Service assigns a block of storage to an instance. The block of storage is a chunk of disk space on virtual disks. Oracle Compute manages virtual disks independently of the instances that use them. When an instance terminates, another instance can replace it and use the virtual disk space allocated to it. This section specifies the operations that can be performed to manage storage components.
