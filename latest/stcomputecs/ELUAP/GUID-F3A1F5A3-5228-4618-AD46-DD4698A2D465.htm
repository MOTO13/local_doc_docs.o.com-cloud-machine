Preface

This guide describes the Oracle Compute Cloud Service REST API for Oracle Public Cloud Machine, which you can use to develop applications and services, in the language of your choice, to interoperate with Oracle Compute Cloud Service.

Audience

This document is intended for tenant administrators and tenant users who use Oracle Compute Cloud Service for Oracle Public Cloud Machine, and who are familiar with the following:

  • The UNIX command line

  • Virtualization technologies

  • Networking and disk storage concepts

  • REST API concepts
