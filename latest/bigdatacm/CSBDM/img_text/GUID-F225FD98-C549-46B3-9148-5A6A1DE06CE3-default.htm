This image shows the Instance Details page of the Oracle Cloud My Services — Create New Oracle Big Data Cloud Service Instance wizard.

At the top of the page is a “Cancel” button, followed by a “train track” diagram that shows progress in the wizard. The first item is “Instance Details.” The second item is “Create Service Instance.” It is selected.

To the right of the diagram is a “Create Service Instance” button. It is surrounded by a box.

The bottom half of the page contains a read-only list of the details that were just entered. The details are described in the surrounding text.
