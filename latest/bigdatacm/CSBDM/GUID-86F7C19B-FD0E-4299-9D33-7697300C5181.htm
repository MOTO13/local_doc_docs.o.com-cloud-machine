Preface

Using Oracle Big Data Cloud Machine describes how to use Oracle Big Data Cloud Machine to store and manage large amounts of data of various data types in the cloud, by using Hadoop HDFS file system and associated services and tools.

Topics

  • Documentation Accessibility

  • Conventions

Audience

Using Oracle Big Data Cloud Machine is intended for administrators and users who want to provision a Hadoop cluster in the cloud and use it to manage big data.
