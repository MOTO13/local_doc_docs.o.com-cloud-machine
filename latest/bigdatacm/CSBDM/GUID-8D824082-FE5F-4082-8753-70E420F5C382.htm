Adding Support for Oracle Audit Vault and Database Firewall

Only Release 12.1.1 of Oracle Audit Vault and Database Firewall Server is supported for use with Oracle Big Data Cloud Machine at this time.

Before installing support on Oracle Big Data Cloud Machine, ensure that Oracle Audit Vault and Database Firewall Server is up and running. The software must be installed on a separate server on the same network as Oracle Big Data Cloud Machine.

You must also have the following information about the Audit Vault Server installation:

  • Audit Vault Server administration user name and password

  • Database service name

  • IP address

  • Port number

To add support for Oracle Audit Vault and Database Firewall: 
  1. Log in to the first NameNode (node01) of the cluster.
  2. Add support for Oracle Audit Vault and Database Firewall: 
    # bdacli enable auditvault
