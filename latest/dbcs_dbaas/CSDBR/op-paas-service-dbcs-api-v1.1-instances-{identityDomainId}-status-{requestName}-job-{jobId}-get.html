View the Job Status of an Operation

get

/paas/service/dbcs/api/v1.1/instances/{identityDomainId}/status/{requestName}/job/{jobId}

Returns the status of a Database Cloud Service instance operation.
  • Request
  • Response

Request

Path Parameters
identityDomainId
Type: string
Required: true
Name of the identity domain for the Database Cloud Service account.
jobId
Type: string
Required: true
Job ID.
requestName
Type: string
Required: true
Name of the request for which you want to view status. Valid values are:
  • create: View the status for a create service instance operation, as described in Create a Service Instance.
  • delete: View the status of a delete instance operation, as described in Delete a Service Instance.
  • control: View the status of a stop, start or restart operation, as described in Stop, Start or Restart a Service Instance or Compute Node.
Header Parameters
Authorization
Type: string
Required: true
Base64 encoding of the user name and password of the user making the request. For more information, see Security, Authentication and Authorization.
X-ID-TENANT-NAME
Type: string
Required: true
Name of the identity domain for the Database Cloud Service account. For more information, see Security, Authentication and Authorization.
