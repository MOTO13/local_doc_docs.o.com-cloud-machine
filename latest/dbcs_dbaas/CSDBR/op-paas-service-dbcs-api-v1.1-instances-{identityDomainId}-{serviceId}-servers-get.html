View a Service Instance's Compute Nodes

get

/paas/service/dbcs/api/v1.1/instances/{identityDomainId}/{serviceId}/servers

Returns information about the compute nodes in a Database Cloud Service instance.
  • Request
  • Response

Request

Path Parameters
identityDomainId
Type: string
Required: true
Name of the identity domain for the Database Cloud Service account.
serviceId
Type: string
Required: true
Name of the Database Cloud Service instance.
Header Parameters
Authorization
Type: string
Required: true
Base64 encoding of the user name and password of the user making the request. For more information, see Security, Authentication and Authorization.
X-ID-TENANT-NAME
Type: string
Required: true
Name of the identity domain for the Database Cloud Service account. For more information, see Security, Authentication and Authorization.

Response

Supported Media Types
  • application/json
202 Response
Accepted. See Status Codes for information about other possible HTTP status codes.
Body
Root Schema : Response Body
Type: array
Title: Response Body
Array of JSON object elements, with one element for each of the service instance's compute nodes.
Items
  • Type: object Response Body Array Item
    Title: Response Body Array Item
Nested Schema : Response Body Array Item
Type: object
Title: Response Body Array Item
Properties
connect_descriptor
Type: string
The connection descriptor for Oracle Net Services (SQL*Net).
connect_descriptor_with_public_ip
Type: string
The connection descriptor for Oracle Net Services (SQL*Net) with IP addresses instead of host names.
created_by
Type: string
The user name of the Oracle Cloud user who created the service instance.
creation_job_id
Type: string
The job id of the job that created the service instance.
creation_time
Type: string
The date-and-time stamp when the service instance was created.
hostname
Type: string
The host name of the compute node.
initialPrimary
Type: boolean
Indicates whether the compute node hosted the primary database of an Oracle Data Guard configuration when the service instance was created.
listenerPort
Type: integer
The listener port for Oracle Net Services (SQL*Net) connections.
pdbName
Type: string
The name of the default PDB (pluggable database) created when the service instance was created.
reservedIP
Type: string
The IP address of the compute node.
shape
Type: string
The Oracle Compute Cloud shape of the compute node.
sid
Type: string
The SID of the database on the compute node.
status
Type: string
The status of the compute node:
  • In Progress: the compute node is being created.
  • Maintenance: the compute node is being stopped, started, restarted or scaled.
  • Running: the compute node is running.
  • Stopped: the compute node is stopped.
  • Terminating: the compute node is being deleted.
storageAllocated
Type: integer
The size in GB of the storage allocated to the compute node. For compute nodes of a service instance hosting an Oracle RAC database, this number does not include the storage shared by the nodes.
