Add an SSH Public Key

post

/paas/api/v1.1/instancemgmt/{identityDomainId}/services/dbaas/instances/{serviceId}/credentials/crednames/{credentialName}

Adds an SSH public key to a Database Cloud Service instance.

  • Response

Type: string
