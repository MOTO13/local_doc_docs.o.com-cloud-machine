URL Structure

Use the following URL structure to access the Database Cloud Service REST endpoints:

https://rest-server:7103/endpoint-path

Where:

  • rest-server is the REST server to contact for your identity domain. Get the host name or IP address of this server from your Oracle Cloud Administrator or Tenant Administrator.

  • endpoint-path is the relative path that defines the endpoint. For a complete list of endpoints, see All REST Endpoints.
