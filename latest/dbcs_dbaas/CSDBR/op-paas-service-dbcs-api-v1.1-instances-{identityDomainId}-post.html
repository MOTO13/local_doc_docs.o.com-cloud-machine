Create a Service Instance

post

/paas/service/dbcs/api/v1.1/instances/{identityDomainId}

Creates a Database Cloud Service instance.

  • Response

Supported Media Types

Properties
customParameters
Type: object customParameters Object
Title: customParameters Object
description
Type: string
Free-form text that provides additional information about the service instance.
edition
Type: string
Required: true

Database edition for the service instance:

  • SE: Standard Edition

  • EE: Enterprise Edition

  • EE_HP: Enterprise Edition - High Performance

  • EE_EP: Enterprise Edition - Extreme Performance

If you specify SE, a Standard Edition Two database is created if you specify 12.1.0.2 for version and a Standard Edition One database is created if you specify 11.2.0.4 for version.

Edition must be Enterprise Edition - Extreme Performance to configure the Database Cloud Service instance as Cluster Database.

level
Type: string
Required: true
Service level for the service instance; use the value PAAS.
parameters
Type: array parameters Array
Title: parameters Array
Required: true
Array of one JSON object that specifies configuration details of the services instance. This array is not required if the level value is BASIC.
serviceName
Type: string
Required: true
Name of Database Cloud Service instance. The service name:
  • Must not exceed 50 characters (15 characters for a service instance that uses Oracle RAC).
  • Must start with a letter.
  • Must contain only letters, numbers, or hyphens.
  • Must not contain any other special characters.
  • Must be unique within the identity domain.
shape
Type: string
Required: true
Desired compute shape. A shape defines the number of Oracle Compute Units (OCPUs) and amount of memory (RAM). Valid shapes include:
  • oc3: 1 OCPU, 7.5 GB memory
  • oc4: 2 OCPUs, 15 GB memory
  • oc5: 4 OCPUs, 30 GB memory
  • oc6: 8 OCPUs, 15 GB memory
  • oc1m: 1 OCPU, 15 GB memory
  • oc2m: 2 OCPUs, 30 GB memory
  • oc3m: 4 OCPUs, 60 GB memory
  • oc4m: 8 OCPUs, 120 GB memory
.

See About Shapes in Using Oracle Compute Cloud Service for more information about shapes.

subscriptionType
Type: string
Required: true
Billing unit. Valid values are:
  • HOURLY: Pay only for the number of hours used during your billing period. This is the default.
  • MONTHLY: Pay one price for the full month irrespective of the number of hours used.
version
Type: string
Required: true
Oracle Database software version; one of: 12.2.0.1, 12.1.0.2 or 11.2.0.4.
vmPublicKeyText
Type: string
Required: true
Public key for the secure shell (SSH). This key will be used for authentication when connecting to the Database Cloud Service instance using an SSH client. You generate an SSH public-private key pair using a standard SSH key generation tool. See Connecting to a Compute Node Through Secure Shell (SSH) in Using Oracle Database Cloud Service.
