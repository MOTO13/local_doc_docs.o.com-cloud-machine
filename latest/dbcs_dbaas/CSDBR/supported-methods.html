Supported Methods

You can create, view, delete and manage Database Cloud Service instances using standard HTTP method requests, as summarized in the following table.

This tables describes the REST methods for managing Database Cloud Service instances.

Method

Description

DELETE

Delete a Database Cloud Service instance.

GET

Retrieve information about all Database Cloud Service instances, a specific instance, or an operation affecting an instance.

POST

Create, stop, start or restart a Database Cloud Service instance.

PUT

Scale a Database Cloud Service instance.
