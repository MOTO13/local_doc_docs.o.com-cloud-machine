Exadata Cloud at Customer

Get Started

Oracle Database Exadata Cloud at Customer combines cloud simplicity, agility, and elasticity with deployment inside your data center to provide full-featured Oracle Databases hosted on Oracle Exadata Database Machine.

Use the following topics to get started, or use the menu to explore everything from what's new to other documentation.

Oracle Database Exadata Cloud at Customer is just one of the offerings that deliver Oracle Cloud services in your data center. Visit cloud.oracle.com/cloud-at-customer to learn more about all of the Cloud at Customer offerings.
