About the REST APIs

Use the REST API for Oracle SOA Cloud Service on Oracle Public Cloud Machine to manage, monitor, and perform operations on SOA services and servers. To manage Oracle Managed File Transfer on Oracle Public Cloud Machine, see the REST API for Oracle Managed File Transfer on Oracle Public Cloud Machine.

APIs

Manage Restore Operations

Restore a service instance, list restore operations, and retrieve the details of a restore operation.
View API Endpoints

Manage Servers

View, add, and remove managed servers. Retrieve server operations and status. View clusters and their servers.
View API Endpoints

Manage Service Backups

Retrieve, create, archive, delete, or list service backups.

View or delete service instances.
