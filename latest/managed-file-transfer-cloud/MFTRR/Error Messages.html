Error Messages

The following table summarizes the error messages that may be returned when you send requests to the Oracle Managed File Transfer Cloud Service REST API and provides a short description and workaround.

Error Code

Error Key

Error Message

Workaround

MFT-5403

MFT_WS_DISCOVERY_COULD_NOT_QUERY_MFT_SOURCES

Discovery service is unable to query MFT Sources for the given application {application} and search Filter {search filter}.

The parameters used for the application and search filter are invalid. Resubmit your request with valid parameters.

MFT-5436

MFT_WS_EVENT_SERVICE_NO_EVENT_FOUND

Error in processing EventService request. No Event found with provided Event Session Id {id}.

No event was found matching the submitted Event Session Id. Resubmit your request with a valid Event Session Id.

MFT-5423

MFT_WS_EVENT_SERVICE_SOURCE_DONT_EXITS_OR_NOT_DEPLOYED

Error in processing EventService request. The source {source} either doesn't exist or not deployed

The event service triggered either does not exist or is not deployed. Deploy the source or check that the name of the source is correct and resubmit your request.
