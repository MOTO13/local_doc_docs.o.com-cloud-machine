About the REST APIs

Use the REST API for Oracle Managed File Transfer Cloud Service to list services, administer metadata and artifacts, discover and trigger sources, and purge old instance and payload data. To manage SOA Cloud Service servers and services, see the REST API for Oracle SOA Cloud Service.
